WritePad Editor

 WritePad is an editor application that allows you to create Web pages easily and quickly.

Steps to open Write Pad application:
  1. Click Start
  2. Click Programs
  3. Click WritePad
Advantages of using a WritePad editor:
  • It allows faster creation of Web pages.
  • It allows easier updation of text or images in WritePad aditor.
  • It is platform independent.
Disadvanteges of using a WritePad editor:
  • It consumes more memory space.
  • It requites to fill-in more details such as meta date for each page.
